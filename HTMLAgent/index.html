General Info

HTML Agent can be found on the google chrome webstore and is created and maintained by Yazaar. The extensions purpose is to bind your webactivity to a webserver for the purpose to make reactive experiences cross domain possible such as currently playing overlays for streamers. One example is my MusicLink project, which can be used together with this extension to bind YouTube, SoundCloud, Monstercat and other music sources directly to a recording/streaming software
